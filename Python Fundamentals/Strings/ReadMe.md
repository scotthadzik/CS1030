 # Advanced Strings
 > ## Strings are just a list of charecters

- Characters can be: 
    - Letters  ` [A-Za-z] `
	- Numbers  ` [0-9] `
	- Symbols  ` @ $ # ~ `
	    - [Unicode Characters Site](https://pythonforundergradengineers.com/unicode-characters-in-python.html)
	- Spaces   `_ _`
	- Escape Character `\n \r \t`

> ## String Indexing
